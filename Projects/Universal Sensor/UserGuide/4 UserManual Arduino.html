4 User Manual Arduino

4.0 Introduction

This Chapter contains the user manaual for thel Lora Sensors supporting down-stream communication for commands coming from the gateway. Please remember that the gateway itself does not issue these commands but they are received by the gateway from ttn console of your own apps.

The following section is made separately for the Arduino sensor MCU's and contains the following supported commands

4.1 Arduino Commands

The Arduino devices supporot the following downstream commands:

  • 0x84 C0
  • 0x86 C4 08
  • 0x88 C8 00 10
