The Universal Sensor Sketch

Introduction

These documents contain the documentation for the Universal Sensor code. There are LoRa sensors in different types and sizes. Apart from the sensor itself they differ mostly in power usage amd local storage capacity. There are sensor in the collowing types:

  • Based on Ardino Pro-Mini
  • Based on ESP8266 or ESP32 technology
  • Base on Cube-Cell (Heltec)

 

Sensors based on Arduino

This is the oldest form of Lora Sensors that I used. For Arduino most sensors were already available, so the focus was on finding a Lora stack that worked in the available amount of memory and did not consume too much power.

Sensors based on Cube-Cell

In october 2019 Heltec started with the Cube-Cell sensor initiative. For this they dod nof only offer an Arduino IDE interface with development board but also delivered a "capsule" device with internal antenna, interface for a battry and solarcells. It could work for several months on a battery and would have more memory available for sensor development.

 

Not Used Technology

Of course there are more mcu's available to build Lora sensors with. Some even have far better specs than the Arduinos. However, power usage of the Arduino is minimal and LoRa sensors do not need much coding and/or power so the Arduino was used for sensors for a long time.

Please look on conrad or Aliexpress to find more mcu's supported.
