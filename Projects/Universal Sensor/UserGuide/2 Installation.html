Universal Sensor

 

Introduction

This document describes the installation of the Universal Sensor. This project started as a way to use the same source code for several types of sensors where the conifguration was determined by settings in its config.h file.

To install the universal sensor sketch, please copy the sket and its related .ino and .h files ot the sketch directory of your choice. Make sure that you set the sketch directory right in the <File><Preferences> screen in the sktechbook location.
