Hardware Guide

Version: 1.0.1
Date: March 2016

This page deals with the various types of Nodes (hardware) connected to the LoRa network. For each project I try to document the design decisions I have taken (if any), document pin-outs etc. and publish some pictures of the project in its various stages of development.

Structure of this document

This document is structured around hardware families for MCU that support thethingsnetwork.org. For every family one or more nodes and/or sensors based on these nodes are described.
